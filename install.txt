The package contains following three important files:
1.usbmonitor.py -- This is the python script that will run in forever mode to listen for the usb devices events from udev and pass it to notify2 module and give desktop notification
2.usbmonitor.sh -- This is a shell script which will create service for usbmonitor and make the service run on boot
3.usbmonitor_1.0_all.deb -- This is a deb file with all files needed to run
